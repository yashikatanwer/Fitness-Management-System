# 🏋️ Fitness Management System

A responsive **Fitness Management System** developed using **HTML, CSS, and JavaScript**. The application helps users monitor their health by calculating BMI, estimating daily calorie requirements, recommending water intake, and suggesting diet plans. It also includes trainer selection, trainer ratings, and an interactive fitness dashboard.

---

## 📌 Project Overview

The Fitness Management System is designed to provide users with a simple and user-friendly platform for managing their basic fitness information. It combines health calculations with an intuitive interface to encourage users to maintain a healthier lifestyle.

---

## ✨ Features

* 🔐 Login Page
* 👤 User Dashboard
* 🏋️ Trainer Selection
* 📋 Trainer Profile
* ⭐ Trainer Rating System
* 💬 Chat with Trainer (UI)
* 📅 Workout Plan Button
* 📊 BMI Calculator
* 🔥 Daily Calorie Requirement
* 💧 Water Intake Recommendation

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in your web browser.
4. Log in using any valid email format.
5. Explore the dashboard and generate your fitness report.
