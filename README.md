# Fitness Management System

## 📌 Overview

A web-based Fitness Management System developed as a Software Engineering project.

The application helps users:

- Register and Login
- Choose a Personal Trainer
- Calculate BMI
- Estimate Daily Calories
- Calculate Water Intake
- Get Diet Recommendations
- Rate Trainers

---
